Made with Love ❤️ By Aditya Wahyu Santoso

Game ini dibuat oleh ketua kelas 9.11 (sembilan sebelas) yaitu Aditya Wahyu Santoso, saya adalah sepuh programmer yang memiliki pengalaman programming selama 3 tahun, saya mampu mencipatakan Software, Website, Web Application, dan Video Game. Kebetulan karena kelas 9 akan berpisah maka saya bikin game ini sebagai kenang kenangan, saya juga ingin melanjutkan pendidikan saya ke SMK dengan jurusan TKJ karena itulah saya membuat game ini, karena TKJ akan mempelajari networking / jaringan jaringan komputer kayak gitu, jadi saya kepikiran untuk membuat game multiplayer ini, game ini membutuhkan waktu 3 minggu lebih untuk dibuat, dibikin menggunakan Godot Game Engine, menggunakan protokol TCP dan menggunakan teknologi peer to peer LAN. Saya tidak bermaksud untuk menghina siapapun di game ini (maaf jika misalkan merasa dihina) dan maaf kalau mungkin gamenya memiliki konten yang mungkin bagi beberapa orang dianggap tidak ramah
